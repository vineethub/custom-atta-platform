# User Stories

## Epic: Blend Builder

### Story 1

As a customer,

I want to select grains,

So that I can create my own atta blend.

Acceptance Criteria:

* User can select multiple grains
* Grain list is loaded from the database

---

### Story 2

As a customer,

I want to assign percentages to grains,

So that I can customize my blend.

Acceptance Criteria:

* Total must equal 100%
* Validation prevents invalid combinations

---

### Story 3

As a customer,

I want to see nutrition values instantly,

So that I understand the benefits of my blend.

Acceptance Criteria:

* Protein displayed
* Fiber displayed
* Calories displayed
* Updates in real time

---

### Story 4

As a customer,

I want to see pricing instantly,

So that I know the cost before checkout.

Acceptance Criteria:

* Price recalculates on percentage change
* Supports different package sizes

---

## Epic: Ordering

### Story 5

As a customer,

I want to add my blend to cart,

So that I can purchase it.

---

### Story 6

As a customer,

I want to checkout securely,

So that I can place my order.

---

## Epic: User Accounts

### Story 7

As a customer,

I want to view my order history,

So that I can reorder previous blends.

---

### Story 8

As a customer,

I want to save favorite blends,

So that I can order them again easily.

---

## Epic: Admin

### Story 9

As an admin,

I want to manage grain inventory,

So that stock remains accurate.

---

### Story 10

As an admin,

I want to update order status,

So that customers receive order updates.

Order States:

* Pending
* Confirmed
* Processing
* Packed
* Shipped
* Delivered
* Cancelled

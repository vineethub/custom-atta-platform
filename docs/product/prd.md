# Product Requirements Document (PRD)

## Product Name

Personalized Atta Platform

## Problem

Consumers cannot easily create atta blends that match their specific nutritional goals.

## Target Users

### Fitness Enthusiasts

Goals:

* High protein
* Muscle gain

### Health Conscious Families

Goals:

* Better nutrition
* Multigrain alternatives

### Diabetic Consumers

Goals:

* Controlled grain composition

### Senior Citizens

Goals:

* High fiber
* Easier digestion

## MVP Goals

1. Allow users to create custom atta blends.
2. Calculate nutritional information in real time.
3. Calculate pricing in real time.
4. Enable online ordering.
5. Provide admin order management.

## Out of Scope (MVP)

* Mobile App
* AI Recommendations
* Referral Program
* Loyalty Points
* Multi-language Support

## Functional Requirements

### Blend Builder

Users can:

* Select grains
* Set grain percentages
* View blend composition

Validation:

* Total percentage must equal 100%

### Nutrition Calculator

System must calculate:

* Protein
* Fiber
* Calories

### Price Calculator

System must calculate:

* Cost per kg
* Total order cost

### Checkout

Users can:

* Enter shipping address
* Pay online
* Place orders

### Admin Dashboard

Admin can:

* Manage grains
* Manage pricing
* Manage orders
* Update order status

## MVP Success Criteria

* First 10 successful orders
* Order fulfillment workflow operational
* Customer feedback collected
